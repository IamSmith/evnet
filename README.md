# evnet

ZeroMQ based event dispatcher

[![build status](https://secure.travis-ci.org/serby/evnet.png)](http://travis-ci.org/serby/###)

## Installation

### ZeroMQ Installation

#### Mac OsX

      brew install zmq

#### Debian derivatives; Ubuntu etc

      sudo apt-get install libzmq1
      sudo apt-get install libzmq-dev

## NPM Install

      npm install evnet
